## Exercices sur la fonction `arange`

### Exercice 1 : Série de nombres de 0 à 9
+++ Enonce
Utilise `arange` pour créer un tableau contenant les nombres de 0 à 9 (inclus).
+++ Code
```python
import numpy as np
tableau = np.arange(10)
print(tableau)
```
